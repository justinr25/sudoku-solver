# Sudoku Solver
A responsive web-based Sudoku solver and puzzle generator with visual backtracking animation that demonstrates how the algorithm works step by step.

## Demo
![Demo of Sudoku Solver](assets/sudoku-solver-demo.gif)

## Usage
1. Open `index.html` in your web browser
2. Either input your own puzzle or click "Generate Puzzle" for a random one
3. Click "Solve" to watch the backtracking algorithm in action
4. Use "Reset" to clear the board and start over
